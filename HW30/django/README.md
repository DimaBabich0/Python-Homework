# Разработка серверных приложений
## Инструкция с клонирования
- Копировать код (клонировать проект)
- Обновить (создать) виртуальное окружение
  * Открыть директорию, выше за проекта (django)
  * `python -m venv django`
  * Проверить появление доп. директорий
- Активировать виртуальное окружение
  * Переходим в директорию проекта: `cd django`
  * `./Scripts/Actibate.ps1`
  * Проверяем изменение консоли
- Обновляем зависимости проекта (пакеты): `pip install Django`, проверяем в появлении директорий в `Lib`
- Переходим в Django-проект (project221): `cd project221`
- Выполняем миграцию: `python manage.py migrate`
- Запускаем сервер: `python manage.py runserver`
- Переходим на сайт: http://127.0.0.1:8000/ или http://localhost:8000/
